# Wedding Share 婚禮照片牆 🎉

這是一個專為婚禮設計的即時照片分享與投影系統。賓客可以透過手機上傳照片並發送彈幕祝福，現場投影幕會即時輪播顯示。

## 🚀 快速開始

### 1. 啟動系統
在專案目錄下執行：
```bash
docker-compose up -d
```

### 2. 登入方式 (重要)

#### � 一般賓客 (手機端 - 用來上傳)
賓客只需掃描 QR Code 或輸入以下網址即可上傳照片與發送彈幕：
- **網址**：`http://<您的IP>:5000/Gallery?identifier=default`
- **功能**：上傳照片 / 發送彈幕祝福

#### �️ 投影幕 (現場大螢幕 - 用來輪播)
在連接投影機的電腦開啟：
- **網址**：`http://<您的IP>:5000/Gallery?identifier=default&mode=slideshow`
- **功能**：自動輪播照片 / 顯示彈幕 / 顯示 QR Code

#### ⚙️ 管理員後台 (Admin - 用來設定)
新郎新娘或工作人員管理照片與設定：
- **網址**：`http://<您的IP>:5000/Account/Login`
- **預設帳號**：`admin`
- **預設密碼**：`admin`
- **功能**：審核照片 / 下載所有照片 / 修改系統設定

---

## 🔧 設定說明 (.env)

請修改 `.env` 檔案中的 `BASE_URL` 為您的電腦 IP 位址：
```bash
BASE_URL=http://192.168.1.107:5000
```
*(注意：BASE_URL 只要填到 :5000 即可，不要包含後面的 /Gallery...)*

## ✨ 特色功能
- **隱私上傳模式**：賓客僅能上傳，無法瀏覽他人照片，確保隱私。
- **全螢幕輪播**：投影模式自動全螢幕寬度顯示，完美展示照片細節。
- **彈幕系統**：支援多色彈幕，獨家 **防遮擋技術** (Always On Top)，確保祝福語不被標題列遮住。
- **極簡賓客介面**：賓客端自動隱藏 QR Code 與數據統計 (Total/Approved)，介面乾淨無干擾。
- **照片輪播**：Smooth 的淡入淡出效果，自動展示 QR Code。
- **繁體中文**：全介面繁體中文化。
- **Docker 部署**：一鍵啟動，輕鬆部署。

## ⚠️ 常見問題
1. **無法連線？** 請確認電腦防火牆已允許 Port 5000，並確認手機與電腦在同一 WiFi 下。
2. **賓客看不到照片？** 這是正常的！為了保護隱私，賓客端 (Upload 介面) 預設會隱藏照片牆，照片只會在投影端 (Slideshow) 顯示。
3. **管理員密碼？** 預設帳號密碼皆為 `admin`，Default 相簿預設無 Secret Key (無需密碼介面)。
4. **沒有彈幕框？** 請確認您是進入 `?identifier=default` 頁面 (手機端) 或 `mode=slideshow` (投影端)。
5. **怎麼取得 QR Code？** 開啟投影網址 (Slideshow 模式) 會自動輪播 QR Code 投影片。
